The transform property in CSS is used to rotate, change size, and position of an HTML element without affecting other elements layout(like margin or postion) The transform property allows you to move (translate), rotate, scale (resize), or skew (tilt) an element on the page. 1)translate(x, y):Moves the element left/right or up/down example:transform: translate(50px, 20px); 2)rotate(angle):Rotates the elements about X or Y or Z axis example:transform: rotate(45deg); or rotateX(45deg) or rotateY(45deg) or rotateZ(45deg) 3)scale(x, y):Changes size (x = width, y = height) example:transform:scale(1.5, 1.5); 4)skew(x, y):Tilts the element example:transform: skew(20deg, 10deg); 5)matrix():Combination of all the above (advanced) transform: matrix(...)
