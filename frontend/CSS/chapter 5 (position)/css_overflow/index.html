CSS overflow property controls how extra content is handled when it overflows its container. In simple words it decides whether extra content is visible, hidden, or scrollable. 1)oveflow:visible -(default)Extra content is shown outside the container. 2)overflow:auto -Shows scrollbars only when required. 3)overflow:scroll-Always shows scrollbars, even if not needed. 4)oveflow:hidden:Extra content is cut off and not visible.
