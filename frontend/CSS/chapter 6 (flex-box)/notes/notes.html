Chapter-06 – CSS Flexbox

What is CSS Flexbox?

CSS Flexbox (Flexible Box) is a one-dimensional layout model used to design one-Dimentional flexible and responsive layouts.

  • It works in one direction at a time (row or column)
  • Makes alignment and spacing easier
  • Helps in building responsive layouts

One-Dimensional Layout (Flexbox)

Flexbox controls layout in one direction at a time — either row (horizontal) or column (vertical).

Figure: One-Dimensional Layout in Flexbox

Flex Container & Flex Items

When display: flex is applied to an element:

  • The element becomes a flex container
  • Its direct children become flex items
.container {
  display: flex;
}
Note: Only direct children become flex items. Grandchildren are not affected unless their parent is also a flex container.

Main Axis & Cross Axis

  • Main Axis → Default is horizontal (row)
  • Cross Axis → Perpendicular to the main axis
  • Axis direction depends on flex-direction

Figure: Main Axis and Cross Axis in Flexbox

Flex Container Properties

Flex container properties are applied to the parent element (the element on which display: flex is used). These properties control the layout, alignment, and spacing of flex items collectively.

Flex container properties:
  1. flex-direction
  2. justify-content
  3. align-items
  4. flex-wrap
  5. align-content

flex-direction Property

Defines the direction of the main axis.

.container {
  flex-direction: row;
}
  • row (default)
  • row-reverse
  • column
  • column-reverse

justify-content Property

Aligns flex items along the main axis.

.container {
  justify-content: center;
}
  • flex-start
  • center
  • flex-end
  • space-between
  • space-around
  • space-evenly

align-items Property

Aligns-items align the items along the cross axis.

.container {
  align-items: center;
}
  • stretch (default)
  • center
  • flex-start
  • flex-end
  • baseline

flex-wrap Property

Controls whether flex items wrap onto multiple lines.

.container {
  flex-wrap: wrap;
}
  • nowrap (default)
  • wrap
  • wrap-reverse

align-content Property

Aligns multiple rows of flex items along the cross axis.

Important: align-content works only when flex-wrap: wrap is used. 
.container {
  align-content: space-between;
}

Flexbox vs Grid

  • Flexbox → One-dimensional layout
  • Grid → Two-dimensional layout

Figure: Main Axis and Cross Axis in Flexbox

Flex Item Properties

Flex item properties are applied to individual flex items (direct children of a flex container). These properties control the size, order, and alignment of each flex item.

Flex item properties:
  1. align-self
  2. flex-grow
  3. flex-shrink
  4. order

align-self Property

Let’s see what this looks like. You’ll apply align-self to two squares, and for the rest apply align-items: center and flex-direction: row.

flex-grow Property

flex-grow controls how much a flex item should grow relative to other flex items when extra space is available.

Default: flex-grow: 0
If there is no extra space, flex-grow has no effect.

flex-shrink Property

flex-shrink controls how much a flex item should shrink when there is not enough space in the container.

Default: flex-shrink: 1

Prevent Shrinking

.box {
  flex-shrink: 0;
}
Item will not shrink even if the container is smaller.
Note: The shorthand property flex is used for flex-grow, flex-shrink, and flex-basis.

flex-grow vs flex-shrink (Summary)

flex-grow
  • Works when extra space is available
  • Grows items to fill empty space
flex-shrink
  • Works when space is insufficient
  • Shrinks items to fit container

Short Exam Definition

CSS Flexbox is a one-dimensional layout module that helps in creating flexible and responsive layouts.
